Run jobs, crews, and billing from one place.

LinkCrew gives owners a clean operating layer for real-time field updates, site photos, supply requests, and invoicing without chasing the crew all day.

Calendar Sync
Google and Outlook should be staged, not rushed
Start with read-in visibility from Google Calendar and Microsoft Outlook, then decide if outbound sync is worth the added conflict complexity.
Recommended Rollout
1. Connect Google or Microsoft. 2. Import external calendar events as visibility only. 3. Let LinkCrew create outbound events. 4. Add two-way edits only if conflicts are manageable.
This keeps the web planner stable before mobile apps inherit the same scheduling rules.
